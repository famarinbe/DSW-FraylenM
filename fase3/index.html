Mi proposito es aprender manejar etiquetas semánticas, CSS Grid para diseñar sitios web

Las etiquetas semánticas se definen como aquellas partes que le dan significado a las partes de un documento. Si lo relacionamos con HTML5 las etiquetas semánticas son aquellas que determinan la parte del sitio web y la información que se de proporcionar. Ejemplo: la etiqueta HEARD se refiere al encabezado de la pagina y los elementos que van en este sector.









Las etiquetas semánticas segun Según Bernal, D., Castro, A., & González, J. La implementación de la web semántica ha sido escasa debido a la falta de conocimiento de las tecnologías que la componen. Ellos describen la Web Semántica y sus antecedentes, posteriormente se presentan las tecnologías que la soportan de una forma sencilla y gráfica, para finalmente reflexionar sobre el papel del usuario final en la Web Semántica y sobre el futuro de esta. Referencia Bernal, D., Castro, A., & González, J. (2014). Web semántica, más de una década de su aparición. Las etiquetas semánticas las entiendo como aquellas que direcionan que es lo que se va a trabajar.
